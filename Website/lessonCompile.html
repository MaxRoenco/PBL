Rules:

  • Put each element on a different line.
  • Start a line with "#" to create a title element.
  • Start a line with "##" to create a subtitle element.
  • Start a line with "$" and then follow it with a link for an image.
  • Don't start a line with anything for a paragraph.
  • Make empty newlines for line breaks.
  • if you want a paragraph to start with "$" or "#" without making a title or image element, you can start it with "\" and then follow it with the symbol you want.
  • for a code segment, start a line with a "`" write your code and close it with the same symbol, if you don't close it, unexpected results might happen.

Example:

# Title 1
This is a paragraph, lorem ipsum dolor I like your shirt's color.
$./localImage.png
$https//:www.example.com/publicImage.png
## Subtitle 2
\# this is a paragraph and it will appear having the symbol # at the beggining because I used "\"
`console.log("this is a code segment");
for(let i = 0; i < 5: i++) {
console.log(i);
}`
